## The Question That Changed It

What if a test told me the highlight disappeared before I even opened the browser?

---

## The Reframe

- A test is an executable specification.
- If I write it first, I must define done clearly.
- If behavior regresses, feedback is immediate and local.

---

## Simpler Way to Say It

I had spent too much time explaining the bug to myself and not enough time defining the behavior clearly.

That was the pivot point from manual confidence to specification-driven confidence.
